# ⚜ Highlighted Projects ⚜

## Pump.fun Revamp
- Gave **Pump.fun** a fresh UI/UX makeover.  
- Built a custom smart contract with extra features and a **Tax Token (Token 2022)** setup.  
- Hooked in **Raydium CPMM migration** and set up a way to share collected tax back to holders.  

## Raydium & Pumpfun Automation Tools
- **Sniper Bot**: Watches pools and fires off trades instantly via **gRPC/RPC**, validating transactions with **Nextblock, bloXroute, and lilJit**.  
- **Bundler**: Spins up Raydium/Pumpfun pools and buys in block one using **Jito bundles**, with live market-making and profit tracking.  
- **Volume Bot**: Helps manage market caps and liquidity across **Raydium, Pumpfun, and Meteora**, nudging volume to keep things stable and growing.  
- **Maker Bot**: Boosts pool activity by coordinating buys across multiple wallets—works great alongside the Volume Bot.  

## Security & Control Mechanisms
- **Token Freezer**: Whitelist-only controls after launch to manage distribution and block unwanted sells.  
- **Token Vesting & Locking Contracts**: Staking and vesting with flexible rewards, time-based boosts, and custom release plans.  

## Presales, Staking & Game Contracts
- Shipped **IDO/ICO smart contracts**, advanced staking modules, and game asset tools on **Solana** and **Sui**.  
- Backed both **DeFi** and **GameFi** with production-ready smart contracts.
